Class VMware_VCloud_API_OrgGuestPersonalizationSettingsType

Description
Description | Vars (details) | Methods (details)

Represents default guest personalization settings for an organization. These values are applied during guest customization of virtual machines in this organization if their GuestCustomizationSection specifies UseOrgSettings.
