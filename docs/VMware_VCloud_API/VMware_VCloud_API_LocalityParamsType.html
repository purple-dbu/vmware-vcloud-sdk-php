Class VMware_VCloud_API_LocalityParamsType

Description
Description | Vars (details) | Methods (details)

Represents locality parameters. Locality parameters provide a hint that may help the placement engine optimize placement of a Vm and an independent a Disk so that the Vm can make efficient use of the disk.
